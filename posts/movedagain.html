RSS feed change

2025 March 03

I am, once again, changing where I'm blogging. I know, this is getting annoying. But the overhead for blogging the way I was doing, in a fully static site environment, was making me not want to do it.

Anyway, more in a minute, but here's the new rss link:

https://calliope.bearblog.dev/feed/

And the blog itself, if you'd like to browse around:

https://calliope.bearblog.dev/blog/

So, what's going on you might ask? Ok, from beginning to end, here is what my workflow looked like for blogging since Cohost shut down:

  • open a markdown file with the blog's template matter in it
  • write the post
  • add in all the metadata stuff in the template area
  • save and quit
  • open terminal
  • navigate to the site's folder on my desktop or laptop
  • run a node.js command
  • wait for it to reformat the files it needs to format and push the new ones into html from markdown
  • close terminal
  • open github desktop
  • commit the changes to /public
  • push the changes to github
  • close github desktop
  • navigate to my hosting provider in Firefox
  • click into the actual hosting side
  • click into the git version control
  • click into the pull menu (I have tried for months and cannot skip this set of steps with a push, as every tutorial says I should be able to)
  • click the button that finds there's an update
  • click the button that puts the update on the hosting service

Now, that's enough. But, and I cannot stress this enough, I have never published a post and not found an error, typo, or other problem in it afterwards. So I have to go through this entire process at least twice for ever post.

I am tired. I am past the part where I can let myself feel proud that I understand some computing stuff I've always wanted to be able to do. I just want something that works and doesn't make editing a total nightmare.

All my posts are still drafted in markdown files, so they're on my pc in case I have to migrate again for whatever reason. But for now, point your browser, and your eyes, at my third party blog and your rss reader here.

I hope, this summer, to turn this site into a proper landing page, with links to all my stuff. Did you know I have a itch page? Probably not, since nothing on this site links to it! I need a place to allow people to commission tarot readings. And if I'm fighting just to write, I won't do any of that. I haven't done any of that, because I've been fighting just to write.

Thanks for your patience.
